# tabuada-em-go
programa simples de tabuada desenvolvido em Go usando loops for para praticar estruturas de repetiçao.

## sobre o projeto
este programa solicita ao usuario um numero e quantas vezes quer repetir a tabuada, depois exibe a a mutiplicaçao usando um loop for. e um exercicio pratico para entender como funcionam os loops for em Go

## como executar
1. certifique-se de ter o Go instalado
2. clone o repositorio
3. navegue ate a pasta do projeto
4. execute "go run main.go"
5. digite o numero da tabuada desejada
6. digite quantas vezes quer repetir
7. veja o resultado

## tecnologias utilizadas
- Go (Golang)
